Manual de ajuda para melhor uso do sistema de directory

O Presente sistema tem como objetivo proteger os ficheiros de cada diretoria deste modo gerando um hmac e aes de modo assegurar integridade e autencidade

A base de dados foi cifrado com a cifra Advanced Encryption Standard no modo Cipher Block Chaining e chaves de 128 bits (AES-128-CBC)

O Hmac de cada ficheiro foi criado com HMAC-SHA512

O Hash de cada ficheiro foi criado com hash Secure Hash Algorithm 256

Para aceder a base de dados clica no botao gerar senha e esta senha usada para descifrar o ficheiro da base de dados

Todos os ficheiros da sua autoria quando forem modificados serás avisado através de mensagens

Qunado clicares na opção mensagem verás todos seus ficheiros que foram alterados e por quem os alterou
